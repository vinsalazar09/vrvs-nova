# VRVS Nova

Reconstrução do zero da plataforma VRVS.

## Status

Projeto em Etapa 0: organização inicial e documentação formal.

## Princípios

- PWA client-side.
- Mobile-first para iPhone Safari.
- IndexedDB como persistência principal.
- Sem backend obrigatório.
- Sem API obrigatória.
- Sem Python.
- Repo novo separado do legado.
- Estabilidade acima de novidade.

## Documentos formais

Os documentos formais ficam em `/docs`.

## Aviso

O legado não é base de implementação.  
Arquivos antigos devem ser consultados apenas como referência futura, quando explicitamente autorizado.
